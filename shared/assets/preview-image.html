Screenshot Instructions for 1200x630px Social Media Preview

  1. Chrome/Edge: Press F12 → Toggle Device Toolbar (Ctrl+Shift+M) → Set to 1200x630 → Screenshot
  2. Firefox: Press F12 → Responsive Design Mode (Ctrl+Shift+M) → Set 1200x630 → Take screenshot
  3. macOS: Cmd+Shift+4 → Spacebar → Click the preview container
  4. Save as: preview-image.png

Optimal size: 1200x630px (Open Graph standard for Medium, LinkedIn, Twitter, Facebook)
